# 📁 How to download project

```bash

    # Clone repository
    $ git clone https://github.com/viniciusperrone/Ignite-Lab

    # Enter directory
    $ cd event-platform

    # Install all dependencies
    $ npm i
    
    # Start server
    $ cd frontend

```
